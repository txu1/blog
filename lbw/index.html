# 后端技术总结

# 前言

我有一个朋友，他本来是以前端身份参加的工作，结果却被领导说服转后端开发，我问他为什么，他说只要是金子在哪都能发光，接着还给我讲了一个故事。

# 故事

在一个夜黑风高的夜晚，路边暖色的灯光好像在照亮晚归的路人，有一只非常招人喜欢的乌鸦在河边找水喝，它突然发现有一只剩半瓶水的瓶子，于是它就往里面不断地丢石子，水变脏了，乌鸦没有喝到水还烂了嘴。

世上根本就没有过去没有未来只有当下，红玫瑰太妖艳，白玫瑰太寡淡，现在拥有的才是最好的。

世上也根本没有神龙秘籍，因为自己就是秘籍，花若盛开蝴蝶自来，蝴蝶不来花也照开。
